# Smart Analytics

smart-analytics repository

# 설치 및 실행 방법

설치 및 실행방법을 설명한다.

# 패키지 설치

$ npm install
1

# 목업 서버(데이터 수집 서버) 실행

$ npm run server
1

# (webpack-dev-server를 통한) 웹서버 구동

$ npm run client
1

or

$ npm start
1

# 데이터 전송 상태 확인

http://localhost:8080 접속하여 브라우저 콘솔 확인
